Guildlines

Entering grammar

  • Each rule should be in a different line. Do not use bars.
  • The left-hand side of the first rule must be the start symbol.
  • Symbols must be separated with spaces.
  • Do not use arrows to separate the left and right-hand sides of a rule. Use a space.
  • Check out the example for further clarification.

Entering input string

  • String must be in a single line
  • Symbols must be separated by space.
  • Do not add an end marker at the end.
  • Check out the example for further clarification.
